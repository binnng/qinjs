QinJs
=====
这是什么
----------
这是一个非常轻量级的js库、目的是实现手机端(webkit)的jquery free 。

###选择器
	$('div')

###绑定器
	'''javascript
	$('div').on('click',function(){
	
	})
	'''

###css渲染
	'''javascript
	$('div').css('display','none')
	'''

###支持线性调用
	'''javascript
	$('div').css('display','none').css('display','block')
	'''
